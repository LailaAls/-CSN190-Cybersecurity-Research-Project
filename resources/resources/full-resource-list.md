# Complete Resource List

This document contains all academic, technical, and learning resources used throughout the CSN 190 Cybersecurity Project during Fall 2024.

---

## Academic Papers

1. Paulsen, C. (2016). *Small Business Information Security: The Fundamentals (NISTIR 7621r1).* National Institute of Standards and Technology (NIST).
   - Link: https://nvlpubs.nist.gov/nistpubs/ir/2016/nist.ir.7621r1.pdf
   - Summary: This report provides foundational cybersecurity guidance for small businesses, including network security basics and risk management practices. It emphasizes low-cost, practical security controls.
   - Used in: Module 2, Module 4

2. Verizon. (2024). *Data Breach Investigations Report (DBIR).* Verizon Enterprise.
   - Link: https://www.verizon.com/business/resources/reports/dbir/
   - Summary: This industry report analyzes real-world data breaches and highlights common attack methods, including exploitation of open ports and misconfigured services.
   - Used in: Module 7 (Medium Article Research)

---

## Technical Documentation

1. Nmap Official Documentation
   - Link: https://nmap.org/docs.html
   - Summary: Used to understand Nmap scan types, command syntax, service detection, and interpreting scan results.

2. Nmap Scripting Engine (NSE) Documentation
   - Link: https://nmap.org/book/nse.html
   - Summary: Used to learn how vulnerability detection scripts work, including HTTP method tampering and denial-of-service checks.

3. macOS Network Commands Documentation
   - Link: https://ss64.com/osx/
   - Summary: Used to understand commands such as `ifconfig`, `ping`, and basic networking utilities on macOS.
